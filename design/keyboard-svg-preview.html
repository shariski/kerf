Sofle v2

6×4 + 5 thumb · 58 keys

Column-staggered with 5-key thumb cluster per side (2 inner keys extending from bottom row + 3 angled outer keys). Leftmost outer thumb key typically houses the rotary encoder.

Lily58

6×4 + 4 thumb · 58 keys

Column-staggered with 4-key thumb cluster per side, arranged in a relatively straight line with innermost key at a slight angle. No encoder on base PCB.
